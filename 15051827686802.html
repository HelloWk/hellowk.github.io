使用 Siri 控制你的家 补充 - 功能更全的 Homebridge 米家接入插件

  • 9/12/2017 11:19 上午
  • posted in  HOME AUTOMATION

在 使用 Siri 控制你的家 2 - 接入米家的产品 中米家的东西加入到我们的 Homekit 中，但是 homebridge-aqara 有一个不好的地方就是支持的设备很少，像 Button 操作，魔方操作，网关自带夜灯，光线传感器都不能利用。

现在，有个更牛的插件出现了，插件作者在 homebridge-aqara 的基础上完善了其他功能的接入，下面让我们来看一下如何启用这个插件。

更换 homebridge-aqara 插件

卸载原有插件

更换插件之前要卸载原有插件：

SSH 进入树莓派， 运行以下命令卸载插件：

sudo npm uninstall -g homebridge-aqara

安装插件

SSH 进入树莓派， 运行以下命令安装插件：

sudo npm install -g npm i homebridge-mi-aqara

修改配置文件

SSH 进入树莓派， 修改配置文件：

vim ~/.homebridge/config.json

然后在 platforms 中将以下配置信息：

{
    "platform": "AqaraPlatform",
    "fakeLightBulbForLightSwitch": true,
    "sid": [
        "aabbccddeeff"
    ],
    "password": [
        "0123456789abcdef"
    ]
}

修改为：

{
    "platform": "MiAqaraPlatform",
    "sid": [
        "aabbccddeeff"
    ],
    "password": [
        "0123456789abcdef"
    ]
}

清理注册的配件缓存

cd ~/.homebridge/accessories/

mv cachedAccessories cachedAccessories_`date '+%Y%m%d_%H%M%S'`.bak

echo [] > cachedAccessories

重启树莓派

sudo reboot

重启过后去 家庭 App 中重新添加一下即可。
