使用 Siri 控制你的家 3 - 开机自动运行 Homebridge

  • 3/23/2017 21:8 下午
  • posted in  HOME AUTOMATION

目前为止我们可以把米家的东西加入到我们的 Homekit 中了，但是还不完美，可能你会发现断开 SSH 的时候，Homebridge 也停止运行了，这样肯定不是一个合格的服务器。

我们的目的是让他自己在那个角落默默地运行，即使断电重启后也会自动运行 Homebridge 服务。

通过 (init.d) 来开机运行 Homebridge

开机自动运行很简单，只需要加入开机启动脚本即可：

SSH 进入树莓派， 运行以下命令：

sudo vim /etc/init.d/homebridge

粘贴以下内容：

#!/bin/sh
### BEGIN INIT INFO
# Provides: homebridge
# Required-Start:    $network $remote_fs $syslog
# Required-Stop:     $remote_fs $syslog
# Default-Start:     2 3 4 5
# Default-Stop:      0 1 6
# Short-Description: Start daemon at boot time
# Description:       Enable service provided by daemon.
### END INIT INFO

dir="/home/pi"
cmd="DEBUG=* /usr/bin/homebridge"
user="pi"

name=`basename $0`
pid_file="/var/run/$name.pid"
stdout_log="/var/log/$name.log"
stderr_log="/var/log/$name.err"

get_pid() {
    cat "$pid_file"
}

is_running() {
    [ -f "$pid_file" ] && ps `get_pid` > /dev/null 2>&1
}

case "$1" in
    start)
    if is_running; then
        echo "Already started"
    else
        echo "Starting $name"
        cd "$dir"
        if [ -z "$user" ]; then
            sudo $cmd >> "$stdout_log" 2>> "$stderr_log" &
        else
            sudo -u "$user" $cmd >> "$stdout_log" 2>> "$stderr_log" &
        fi
        echo $! > "$pid_file"
        if ! is_running; then
            echo "Unable to start, see $stdout_log and $stderr_log"
            exit 1
        fi
    fi
    ;;
    stop)
    if is_running; then
        echo -n "Stopping $name.."
        kill `get_pid`
        for i in {1..10}
        do
            if ! is_running; then
                break
            fi

            echo -n "."
            sleep 1
        done
        echo

        if is_running; then
            echo "Not stopped; may still be shutting down or shutdown may have failed"
            exit 1
        else
            echo "Stopped"
            if [ -f "$pid_file" ]; then
                rm "$pid_file"
            fi
        fi
    else
        echo "Not running"
    fi
    ;;
    restart)
    $0 stop
    if is_running; then
        echo "Unable to stop, will not attempt to start"
        exit 1
    fi
    $0 start
    ;;
    status)
    if is_running; then
        echo "Running"
    else
        echo "Stopped"
        exit 1
    fi
    ;;
    *)
    echo "Usage: $0 {start|stop|restart|status}"
    exit 1
    ;;
esac

exit 0

保存，退出，运行下面的命令：

sudo chmod 755 /etc/init.d/homebridge
sudo update-rc.d homebridge defaults

现在就可以在开机的时候自动运行 Homebridge 了。你也可以手动的开启服务：

sudo /etc/init.d/homebridge start

使用下面的命令即可查看log 和 error：

tail -f /var/log/homebridge.log
tail -f /var/log/homebridge.err

以上。
