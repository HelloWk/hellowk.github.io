正则表达式

发表于 Feb 18 2016   |   分类于 技术   |  

这是一篇技术文，主要学习下“正则表达式”。

概述

　　正则表达式（Regular Expression，简称 regexp）是一种 描述字符串结构的语法规则，是一个特定的格式化模式，他可以匹配、替换、截取匹配的字符串。

正则表达式语法规则

　　 一个完整的正则表达式由元字符喝文本字符两部分组成，其中元字符就是具有特殊含义的字符，如^ $ \ []等；文本字符就是普通的文本，如字母数字等。

定位符 ^ $ \b \B

　　确定字符所在字符串内的具体方位。

字符 ^

　　匹配输入字符串开始的位置
　　

字符 $

　　匹配输入字符串结尾的位置。
　　　

字符 \b

　匹配一个字边界，也就是指单词和空格间的位置。　

字符 \B

　　匹配非单词边界

字符类 []

　　一个字符类定义一组字符，其中的任一字符均可出现在输入字符串中以便成功匹配。
　　正则表达式区分大小写，如果要忽略大小写，我们可以用方括号表达式“[]”。一个方括号能且只能匹配一个字符，并且只要匹配的字符在方括号内，就表示匹配成功。

选择字符 |

　　该字符可以理解成“或”，例如同样要匹配字符串“hi”不区分大小写，可以使用H|hI|i表示。
　　注意：[]和|的区别在于[]只能匹配单个字符，而|可以匹配任意长度的字符串，如HI|Hi|hi|hI。

连字符 -

　　做开发的都知道变量名只能以字母或下划线开头，如果使用正则表达式来匹配变量名中的第一个字符，就需要写成一下这种形式:[abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ]。这么写无疑是很麻烦的，这时我们可以用连字符“-”来简化正则表达式。使用连字符指定字符列表时，需要指定其实字符和结束字符。

注意：

  1. 字符范围类是遵循字符编码的顺序来匹配的
  2. 注意字符类内部不要有空格，否则会被认为是要匹配一个空格，如[0-9 ]。不仅会匹配所有数字，还会匹配所有空格
  3. 在通常情况下字符“-”指表示一个普通字符，只有在表示范围时才被作为元字符使用。

反义字符 [^]

　　反义字符就是除了字符类中指定的字符以外的任意字符，如果在字符类内部添加了^前缀，即定义了反义字符类。

限定符 ? + * {n} {n,} {n,m}

　　限定符指定在输入中必须存在字符、组或字符类的多少个实例才能找到匹配项。

字符 ?

　　匹配前面的字符零次或一次。

字符 +

　　匹配前面的字符一次或多次。

字符 *

　　匹配前面的字符串零次或多次。

字符 {n}

　　匹配前面的字符串 n 次。

字符 {n,}

　　匹配前面的字符串最少 n 次。

字符 {n,m}

　　匹配前面的字符最少 n 次，最多 m 次。

限定符的用法比较灵活，且语义有一定的重叠性，使用时需要小心，以免发生混淆。
注意：当时用元字符*和?时，由于这些字符可能匹配前面字符或表达式0次，所以他们允许什么都不匹配，如正则表达式/a*/实际上与字符串bcd匹配，该字符还有0个a。

点字符 .

　　在正则表达式中，点字符.可以匹配换行符外的任意一个字符。

转义符 \

　　将特殊字符（如. ? \等）转变为普通字符。

反斜线 \

　　\还有其他的功能，可以显示一些不可打印的字符，指定预定义字符集，定义一些限定符。

表1 反斜线输出不可打印字符

字符　　　　 说明
\a 警报，即 ASCII 的 \ 字符
\b 退格，即 ASCII 的 \<> 字符
\e Escape，即 ASCII 的 \<> 字符
\f 换页符，即 ASCII 的 \<> 字符
\n 换行符，即 ASCII 的 \<> 字符
\r 回车符，即 ASCII 的 \<> 字符
\t 水平制表符，即 ASCII 的 \<> 字符
\xhh 十六进制代码
\ddd 八进制代码
\cx 即 control-x 的缩写，匹配由 x 指名的控制字符，其中 x 是任意字符

表2 反斜线指定的预定义字符集

字符　　　　 说明
\d 任意一个十进制数字
\D 任意一个非十进制数字
\s 任意一个空白字符（空格、换行符、换页符、回车符、水平制表符）,相当于[\f\n\r\t]
\S 任意一个飞空白字符
\w 任意一个单词字符，相当于[a-zA-Z0-9]
\W 任意一个非单词字符

表3 反斜线定义短远的限定符

字符　　　　 说明
\b 单词分分界符，用来匹配字符串中的某些位置
\B 非单词分界符序列
\A 总是能够匹配带搜索文本的起始位置
\z 只匹配字符串的末尾，而不考虑任何换行符
\Z 表示在未指定任何模式下匹配的字符，通常是字符串的末尾位置，或者在字符串末尾的换行符之前的位置
\G 当前匹配的起始位置

括号字符 ()

在正则表达式中，括号字符()有两个作用：

  1. 改变限定符作用的范围
  2. 分组
